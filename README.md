# Kamarkett Fastify + Vercel

Projeto exemplo em Node.js usando Fastify para servir uma página HTML de marketplace.

## Rodar local

```bash
npm install
npm run dev
```

Depois abra:

```text
http://localhost:3000
```

## Subir na Vercel

```bash
npm i -g vercel
vercel login
vercel
```

Para produção:

```bash
vercel --prod
```
